Installation (requires R >=3.0 ):
  • Download the version corresponding to your operating system.

    • For Linux (Package source): LEA_0.2.tar.gz
    • For Windows (binaries): LEA_0.2.zip
    • For OS XÂ Mavericks (binaries): LEA_0.2.tgz

  • open R and install the R package LEA
    • # find the path to the LEA package you downloaded here.
install.packages("path to the package")

# example for Linux
install.packages("Downloads/LEA_0.2.tar.gz", repos = NULL, type = "source")

# example for Mac
install.packages("Downloads/LEA_0.2.tgz", repos = NULL)

# example for Windows
install.packages("Downloads/LEA_0.2.zip, repos = NULL")

  • If your MAC operating system is different from Mavericks, install gfortran and the LEA source package LEA_0.1.tar.gz.
