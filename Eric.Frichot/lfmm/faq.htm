Can I analyse large microsattelite dataset with LFMM ?

Yes (if you consider that you have enough microsattelites to both infer for population structure and test for association with the environment).
There are two possible ways to analyze microsattelite dataset:
  • you can test for correlations between the environmental variable and the microsattelite markers. In this case, there is one row per individual. Each row contains one value per microsattelite (separated by spaces or tabulations): the number of alleles. An example of LFMM format for 3 individuals and 4 microsattelite markers is available: microsatellite_1.lfmm.
  • you can test for correlations between the environmental variable and the microsattelite alleles. In this case, there is one row per individual. For microsattelite marker is coded on A columns if it is composed of A differents alleles. Then, for each column, there is a 1 if this individual has this microsattelite allele for this marker and 0 otherwise (separated by spaces or tabulations). An example of LFMM format for 3 individuals and 4 microsattelite markers is available: microsatellite_2.lfmm. It is the same dataset as microsattelite_1.lfmm but for this purpose.

Can I analyse AFLP data with LFMM ?

Yes. In this case, LFMM tests the correlation between the environmental variable and the presence of the dominant marker. You just need to format to LFMM format with 0 for minor marker and 1 for dominant marker, separated by one or several spaces. There should be no space at the end of a line. One line per individual. 9 or -9 value for missing value. An example of LFMM format for 3 individuals and 4 AFLP markers is available: aflp.lfmm.
