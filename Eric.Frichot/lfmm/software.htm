LFMM software is available in command line version (CL) for Linux and Mac Operating System (OS) or with a Graphical User Interface (GUI) for Windows, Linux and Mac Operating System (OS). It also contains a tutorial, a documentation, perl and R scripts and an example.

Command-line version (CL):

  • For Linux and Mac: LFMM_CL_v1-2.zip

Graphical User Interface version (GUI):

  • Linux 64bits:  LFMM_GUI_Linux64_v1-2.zip
  • Linux 32bits:  LFMM_GUI_Linux32_v1-2.zip
  • Mac 64bits: LFMM_GUI_Mac64_v1-2.zip
  • Windows 64bits:  LFMM_GUI_Win64_v1-2.zip
