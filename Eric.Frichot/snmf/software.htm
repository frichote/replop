sNMF software is available in command line version (CL) for Linux and Mac Operating System (OS). It also contains a tutorial, a documentation and an example.

Command-line version (CL):

  • For Linux and Mac: sNMF_CL_v1.1.zip

Versions
  • 1.1: First main release of sNMF. Major bug fixed for CE for K=1. Command-line options and output files modified.
  • 0.5: Update of the documentation. Minor bugs fixed. Cross-Entropy for K=1. Conversion programs from the ancestrymap, lfmm, ped, and vcf formats to geno format. New options.
  • 0.1: First beta version
